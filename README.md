### Install Guide

$\color{red}\textsf{\Large\kern{0.2cm}\ Important this project must have MQTT broker}$

1.  Install/Download NodeJS or Bun
2.  Dowload this project
3.  Install requirement
    - API
      - Go to directory
        ```bash
        cd ../Project/Service/api
        ```
      - Install requirement:
        <br>
        Bun
        ```bash
        bun install
        ```
        NodeJS
        ```bash
        npm i
        ```
        or
        ```bash
        npm install
        ```
    - Line
