### Install Guide

1. Install/Download NodeJS or Bun
2. Dowload this project
3. Install requirement
   > API
   - go to directory
   ```bash
   cd ../Service/api
   ```
   - Install requirement
   * bun
   ```bash
   bun install
   ```
   - NodeJS
   ```bash
   npm i
   ```
   or
   ```bash
   npm install
   ```
